# 项目结构说明

## 📁 核心文件

### LLM接口层
- **`llm_interface.py`** - LLM接口实现，支持讯飞星火、OpenAI、Anthropic等多种后端

### 动态记忆系统核心
- **`memory_system.py`** - 动态记忆系统核心，包含Prompt学习和四大动态过程（存储、检索、遗忘、更新）
- **`memory_storage.py`** - 动态记忆存储和导出功能

## 🚀 使用示例

### 快速开始
- **`quick_start_xinghuo.py`** - 快速开始脚本（推荐新用户）

### 完整示例
- **`example_dynamic_usage.py`** - 完整的动态记忆系统使用示例，包含所有功能演示

### 工具脚本
- **`process_any_text.py`** - 处理任意文本，自动提取记忆

### 测试脚本
- **`test_xinghuo_connection.py`** - 测试讯飞星火API连接

## 📄 配置文件

- **`README.md`** - 项目文档和使用说明
- **`requirements.txt`** - Python依赖包列表

## 💾 数据文件

- **`dynamic_memories.json`** - 动态记忆存储文件（自动生成）

## 🎯 推荐使用流程

1. **测试连接**: `python test_xinghuo_connection.py`
2. **快速体验**: `python quick_start_xinghuo.py`
3. **完整学习**: `python example_dynamic_usage.py`
4. **处理文本**: `python process_any_text.py`

## 📚 代码组织原则

- **单一职责**: 每个文件职责明确
- **无重复**: 删除功能重复的文件
- **易用性**: 提供简单到复杂的多级示例
- **完整性**: 核心功能完整且自包含
